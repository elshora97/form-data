# المنصة المجانية للباحث الكيميائى الدكتور محمد أشرف أبو عيسي

مشروع React جاهز للتشغيل باستخدام Vite.

## طريقة التشغيل

1. فك ضغط الملف (extract).
2. افتح Terminal جوه فولدر المشروع `enrollment-app`.
3. نفّذ الأوامر دي بالترتيب:

```
npm install
npm run dev
```

4. هيديك رابط زي `http://localhost:5173` — افتحه في المتصفح وهتلاقي الصفحة شغالة.

## تجهيز نسخة للنشر (Deploy)

```
npm run build
```

هيطلعلك فولدر `dist` جاهز ترفعه على أي استضافة (Netlify, Vercel, GitHub Pages...).
- الفورم دلوقتي بيعرض رسالة نجاح بس من غير ما يبعت البيانات لسيرفر — لو عايز تربطه بقاعدة بيانات أو Google Sheet أو API، قولّي وهظبطلك الجزء ده.
